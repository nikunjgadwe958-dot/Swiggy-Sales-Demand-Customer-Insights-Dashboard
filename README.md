## ðŸ—‚ Data Model & Datasets Used
The dashboard is built using the following Excel files:

- **Orders**: Order-level details including order date, city, and sales value  
- **Order_Type**: Classification of orders (veg, non-veg, order category)  
- **Restaurants**: Restaurant-level information such as name, city, ratings, and rating count  
- **Users**: Customer demographics including gender, marital status, and occupation  
- **Menu**: Menu-level mapping between restaurants and food items  
- **Food**: Food item details including category, cuisine, and price  

A relational data model was created in Power BI to enable cross-table analysis.
